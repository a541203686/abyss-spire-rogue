# Abyss Spire Rogue（网页卡牌冒险原型）

一个可直接在浏览器游玩的单人卡牌肉鸽原型，灵感来自《杀戮尖塔 / 卡牌冒险家》。

## 项目亮点

- 回合制卡牌战斗：能量、抽牌、弃牌、格挡、力量、易伤、虚弱
- 多层路线推进：普通战 / 精英战 / 营火 / 商店 / 首领战
- 战后奖励：三选一卡（可跳过）
- 商店系统：买卡、补给、战术研修、剪牌服务（含随机折扣）
- 战斗演出：敌方意图图标、伤害飘字、技能与攻击特效
- 主角固定为怪盗角色（已移除换装系统）

## 本地运行

> 需要一个静态文件服务器（推荐 Python 3）

```bash
cd /path/to/abyss-spire-rogue
python3 -m http.server 8765
```

打开浏览器访问：

- http://127.0.0.1:8765/index.html

## 操作说明

- 点击「开始冒险」进入爬塔流程
- 地图页点击「进入下一层」推进
- 战斗中点击手牌出牌，或点击「结束回合」
- 战后在奖励区选 1 张卡加入牌库（或跳过）
- 商店可消费金币购买卡牌与服务

## 在线优化说明（上线建议）

如果你准备把这个项目长期在线部署（例如 GitHub Pages / Vercel / 腾讯云静态托管），建议优先做下面几件事：

1. **静态资源压缩与指纹化**
   - 将 `styles.css`、`game.js` 做最小化压缩。
   - 文件名加入 hash（如 `game.abcd1234.js`），便于强缓存与版本切换。

2. **缓存策略**
   - HTML 走短缓存（或 `no-cache`），保证更新可见。
   - JS/CSS/图片走长缓存（`max-age=31536000, immutable`）。

3. **图片资源优化**
   - 截图与展示图使用压缩后的 PNG/WebP。
   - 保持首屏必要资源优先加载，非关键图可延迟加载。

4. **首屏体验优化**
   - 将关键样式内联或拆分关键 CSS，减少白屏时间。
   - 避免首屏执行不必要逻辑，先保证可交互再补充动画。

5. **线上质量保障**
   - 增加基础监控：页面加载耗时、JS 报错、资源 404。
   - 每次发布前做一次冒烟检查（开始冒险 -> 进入战斗 -> 结束回合）。
